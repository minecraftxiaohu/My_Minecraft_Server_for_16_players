This Minecraft server is based on the file built after executing the "java -jar forge-1.12.2-14.23.5.2859-installer.jar -install" command from the 1.12.2 installer file downloaded from the Forge official website. And use an exe file to run the server (exe file and website are not provided for now). Thanks to minecraft.net and minecraftforge.net for this!
